## Requerimentos
Java Virtual Machine (JVM)

## Configure
Adicione o caminho da pasta "pfh" na variável de ambiente "PATH" e no Unix-like dê a permissão para executar binários para a pasta "pfh" executando o comando "chmod +x [pfhdir]"

## Execute
Use o comando "pfh" para mostrar as opções disponíveis
